Attribute Selector
Meaning / Definition:
  • The [attribute] selector is used to select elements with the specified attribute.
    		•
    Scenario / Use : Select and style every element that are using same attributes.
    Syntax : elementName[attribute] {css properties : values ; }
    Example : elementName[attribute] | p[id] { text-decoration : underline ; }
