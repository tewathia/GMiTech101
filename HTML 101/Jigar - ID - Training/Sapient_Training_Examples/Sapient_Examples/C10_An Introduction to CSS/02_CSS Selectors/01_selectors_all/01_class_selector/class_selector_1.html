Class Selector
Meaning / Definition:
  • The .class selector selects and styles all elements with the specified class.
  • This selector targets on particular “CLASS” Name defined by us. [Customize Class name].
  • Class is generic/general which usually applies to one or more similar propety base objects/elements.
  • The Class selector always start with DOT / . sign.

    Scenario / Use : From bunch of 5 paragraphs, 2 will be of blue color & rest 3 of red color.
    Syntax : .className {css properties : values ; }
    Example : .className | .redText { color:#FF0000 ; }
