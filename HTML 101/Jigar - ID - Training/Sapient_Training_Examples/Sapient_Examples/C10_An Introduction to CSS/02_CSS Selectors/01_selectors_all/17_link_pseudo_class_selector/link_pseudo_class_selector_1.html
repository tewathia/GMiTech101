Link Pseudo Class Selector
Meaning / Definition:
  • This selector is used to set/modify the <A href> links tags & its stages like :link, :visited, :hover, :active etc.
  • Its advisable to follow the below order :
    1. a:link { } - The :link selector is used to select unvisited links.
    2. a:visited { } - The :visited selector is used to select visited links.
    3. a:hover { } - The :hover selector is used to select elements when you mouse over them. Tip: The :hover selector can be used on all elements, not only on links.
    4. a:active { } - The :active selector is used to select and style the active link; A link becomes active when you click on it.
    5. ** input:focus { } = Selects the input element which has focus.
    • Scenario / Use : Wants to customize the by-default link properties. [check w3cschools link styling and create.]
    Syntax : a:state { } {css properties : values ; }
    Example : a:link { } | a:link { text-decoration:none ; }
