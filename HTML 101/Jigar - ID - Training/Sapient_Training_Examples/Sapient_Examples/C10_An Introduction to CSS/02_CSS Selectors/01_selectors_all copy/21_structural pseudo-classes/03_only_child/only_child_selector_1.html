only-child() Selector
Meaning / Definition:
  • The :only-child selector matches every element that is the only child of its parent.
    		•
    Scenario / Use : Want to Set the properties for an element (h1 or p) that is only child of parent.
    Syntax : :only-child {css properties : values ; }
    Example : elementName:only-child | p:only-child{ background-color:#12FF34 ; }
