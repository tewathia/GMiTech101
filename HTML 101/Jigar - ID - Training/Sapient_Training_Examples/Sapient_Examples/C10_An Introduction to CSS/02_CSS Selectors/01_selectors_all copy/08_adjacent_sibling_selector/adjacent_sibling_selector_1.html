Element+Element / Adjacent Sibling Selector
Meaning / Definition:
  • This is referred to as an adjacent selector. It will select only the element that is immediately preceded by the former Element.
  • The element+element selector is used to select elements that is placed immediately after (not inside) the first specified element.
    		•
    Scenario / Use : Select and style every <li> element that are placed immediately after <ol> element.
    Syntax : elementName+elementName {css properties : values ; }
    Example : elementName+elementName | div+li { text-decoration : underline ; }
