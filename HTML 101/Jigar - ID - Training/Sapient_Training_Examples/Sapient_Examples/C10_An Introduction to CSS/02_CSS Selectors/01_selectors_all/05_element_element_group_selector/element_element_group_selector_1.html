Element,Element / Group / Compound Selector
Meaning / Definition:
  • Also known as "group" or "compound" selector
  • This selector targets Group/Bunch of Elements/Tags.
  • This selector helps to style several elements, separate each element name with a comma.
  • Helps to style/format more than 1 element/tag/classes/ids.
  • * Known as CSS Optimization Technique Selectors as it saves lots of time and code.
    		•
    Scenario / Use : Want to apply same formatting's to P,LI,H1 and class defined in html page.
    Syntax : elementName,elementName {css properties : values ; }
    Example : elementName,elementName,.className | p,li,h1 { font-size:14px; }
