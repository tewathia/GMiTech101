Universal * Selector
Meaning / Definition:
  • The universal/* selector selects all elements.
  • The universal/* selector can also select all elements inside another element.
    		•
    Scenario / Use : To Selects all elements inside another element. Want to select all Paragraphs inside Divs.
    Syntax : element * {css properties : values ; }
    Example : element * | div * { background-color : #09F; }
