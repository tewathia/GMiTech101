nth-child(n) Selector
Meaning / Definition:
  • The :nth-child(n) selector matches every element that is the nth child, regardless of type, of its parent.
  • n can be a number, a keyword, or a formula.
    		•
    Scenario / Use : Want to Set the properties for every p or h1 element occurs at specific position of any parent element.
    Syntax : elementName:nth-child(n) {css properties : values ; }
    Example : elementName:nth-child(1) | h1:nth-child(1) { color:#12FF34 ; }
