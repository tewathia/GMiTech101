first-child Selector
Meaning / Definition:
  • The :first-child selector is used to select the specified selector, only if it is the first child of its parent.
    		•
    Scenario / Use : Want to Set the properties for every p or h1 element occurs at first position of any parent element.
    Syntax : elementName:first-child {css properties : values ; }
    Example : elementName:first-child | h1:first-child { color:#12FF34 ; }
