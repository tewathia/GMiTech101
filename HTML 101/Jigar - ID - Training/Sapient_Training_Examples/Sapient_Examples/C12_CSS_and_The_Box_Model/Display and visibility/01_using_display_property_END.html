CSS display Property / using display property

The display property defines how a certain HTML element should be displayed.

1. inline

This is default. The element is displayed as an inline-level element (like span) / in a same line.

2. none

The element will not be displayed at all.

3. block

The element is displayed as a block-level element (like paragraphs and headers).

4. inline-block

The element is placed as an inline element (on the same line as adjacent content), but it behaves as a block element.

inline-block elements gets some spaces inbetween.

5. list-item

The element is displayed as a list-item, which means that it has a bullet in front of it.
