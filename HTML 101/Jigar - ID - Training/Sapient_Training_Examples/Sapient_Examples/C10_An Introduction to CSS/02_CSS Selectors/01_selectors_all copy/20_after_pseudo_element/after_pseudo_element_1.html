:after pseudo element selector
Meaning / Definition:
  • The :after selector inserts content after the content of the selected element(s).
  • Use the content property to specify the content to insert.
    		•
    Scenario / Use : Want to insert some content after every p or h1 element's content:
    Syntax : elementName:after {css properties : values ; }
    Example : elementName:after | elementName:after { content:"Read this important text inserted "; }
