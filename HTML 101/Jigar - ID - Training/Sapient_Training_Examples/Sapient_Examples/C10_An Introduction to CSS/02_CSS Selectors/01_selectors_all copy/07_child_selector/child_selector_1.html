Element>Element / Child Selector
Meaning / Definition:
  • The element>element selector is used to select elements with a specific parent.
  • This selector targets on only direct children ie. Elements that are not directly a child of the specified parent, are not selected.
    		•
    Scenario / Use : Select and style every <li> element where the immidiate parent is a <ol> element.
    Syntax : elementName>elementName {css properties : values ; }
    Example : elementName>elementName | ol li { text-decoration : underline ; }
