1. An image sprite is a collection of images put into a single image.
  2. A web page with many images can take a long time to load and generates multiple server requests, image sprites are used to minimize server requests.
  3. With the help of image sprite we can create navigation.
  4. background:url(02_icons_navsprites.png) 0 0; - Defines the background image and its position (left 0px, top 0px)
  5. transparent.gif is used because with IMG tag SRC attribute is must
