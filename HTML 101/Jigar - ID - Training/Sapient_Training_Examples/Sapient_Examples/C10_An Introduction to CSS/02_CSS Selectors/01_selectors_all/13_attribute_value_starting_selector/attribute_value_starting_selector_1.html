Attribute|=Value Selector
Meaning / Definition:
  • The [attribute|=value] selector is used to select elements with the specified attribute starting with the specified value.
  • The value has to be a whole word, either alone, like class="yellow", or followed by a hyphen( - ), like class="yellow-text".
    		•
    Scenario / Use : Select and style elements, where the attribute's value starts with specific word.
    Syntax : elementName[attribute|=value] {css properties : values ; }
    Example : elementName[attribute|=value] | p[class|=yellow] { border : 1px solid red ; }
