Id Selector
Meaning / Definition:
  • The #id selector selects and styles all element with the specified id.
  • This selector targets on unique “ID” Name defined by us. [Customize Id name].
  • Id simply meance identification; which is always unique.
  • The Id selector always start with HASH / # sign.
    		•
    Scenario / Use : From bunch of 5 paragraphs, each one is having unique or different properties.
    Syntax : #idName {css properties : values ; }
    Example : #idName | #redText { color:#FF0000 ; }
