Pseudo Element Selector
Meaning / Definition:
  • This selector targets on First Letter / First Line / First Child of any Element/Tag.
    		•
    Scenario / Use : Want to customize the properties of :first-letter / :first-line / :first-child of any Tag i.e. Paragraph.
    Syntax : elementName:first-letter { css properties : values ; }
    Example : elementName:first-letter { } | p:first-letter { font-size : 50px; }
