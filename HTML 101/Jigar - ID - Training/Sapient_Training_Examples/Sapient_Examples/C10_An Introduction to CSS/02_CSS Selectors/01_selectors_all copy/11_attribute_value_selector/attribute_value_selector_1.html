Attribute=Value Selector
Meaning / Definition:
  • The [attribute=value] selector is used to select elements with the specified attribute and value.
    		•
    Scenario / Use : Select and style every element that are using same attributes value.
    Syntax : elementName[attribute=value] {css properties : values ; }
    Example : elementName[attribute=value] | p[class=yellowText] { text-decoration : underline ; }
