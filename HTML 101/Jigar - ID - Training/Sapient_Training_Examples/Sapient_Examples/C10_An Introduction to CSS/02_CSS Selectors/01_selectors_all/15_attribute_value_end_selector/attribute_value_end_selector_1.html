Attribute$=Value Selector
Meaning / Definition:
  • The [attribute$=value] selector is used to select elements with the specified attribute ends with the specified value.
  • The value has to be a whole word, either alone, like class="yellow", or followed by a hyphen( - ),underscore ( _ ) like class="yellow-text", class="yellow_text".
    		•
    Scenario / Use : Select and style elements, where the attribute's value ends with specific word.
    Syntax : elementName[attribute$=value] {css properties : values ; }
    Example : elementName[attribute$=value] | p[class$=text] { border : 1px solid red ; }
