Element / Tag / Type Selector
Meaning / Definition:
  • The element / tag / type selector selects all elements with the specified element name.
    		•
    Scenario / Use : This selector targets on particular tag of HTML page. Wants to select all Paragraphs and assing formats.
    Syntax : elementName {css properties : values ; }
    Example : elementName | p { font-size:14px; }
