Attribute~=Value Selector
Meaning / Definition:
  • The [attribute~=value] selector is used to select elements with an attribute value containing a specified word.
    		•
    Scenario / Use : Select and style every element that are using attributes value containg specific word.
    Syntax : elementName[attribute~=value] {css properties : values ; }
    Example : elementName[attribute~=value] | p[class~=yellow] { border : 1px solid red ; }
